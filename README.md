An ArcGIS JavaSCript Application - UI
=====================================

A responsive UI (bootstrap) taylored for municipal GIS applications built with ArcGIS JavaScript API (3.11)


Demo 
----

http://gis.wirapids.org/CityViewer-AMD/#

Features
--------
+ Map
    + Basemap
    + Aerial
+ Navigation
  + Pan
  + Zoom
  + Zoom to Parcel
  + GPS
+ Identify
  + Parcel
  + Utilities
+ Search
  + Owner
  + Address
  + Parcel No.
+ Measure
+ Buffer
+ Create Mail Labels
+ Print
