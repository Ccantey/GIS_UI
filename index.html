About City Viewer

Wisconsin Rapids City Viewer was developed by Chris Cantey, GIS Specialist.

This application was built on top of the Bootstrap front-end framework to create a browser-agnostic application, meaning this app is available across all browsers and all browser sizes (including mobile).

The data that drives the app was produced by the City of Wisconsin Rapids and stored inside of an ESRI LocalGovernment Information Data Model. Interaction with the data, including maps, geometry and calculations are served up using ArcGIS Server and REST endpoints called through the ArcGIS for JavaScript API.

The data is expected to be the most recent available to GIS.
